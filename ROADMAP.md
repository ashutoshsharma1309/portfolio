# Roadmap

Planned and considered improvements. Nothing here is a commitment — it's a
public scratchpad for where the portfolio could go next.

## Near term
- [ ] Add live demo + source links to project cards (data already supports them)
- [ ] Wire up canonical URL, `og:url`, `robots.txt`, and a `sitemap.xml`
- [ ] Real project screenshots in the card covers (gradient fallback today)
- [ ] Confirm and surface a single canonical contact email

## Medium term
- [ ] Light/dark room lighting toggle
- [ ] Subtle ambient audio with an explicit, accessible mute control
- [ ] First-visit guided tour highlighting interactive objects
- [ ] Visitor analytics (privacy-friendly, cookieless)

## Exploratory
- [ ] Draggable free-look camera mode (with a "reset view" affordance)
- [ ] Per-project deep-dive routes for shareable links
- [ ] Localization (i18n) of panel copy

See the [CHANGELOG](CHANGELOG.md) for what's already shipped.
